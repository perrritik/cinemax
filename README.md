# CINEMAX

Кинострим-сайт с тёмным кино-стилем. 5 страниц + настраиваемый плеер на каждой.

## Структура файлов

```cinemax/
├── index.html          # Главная: баннер (hero), слайдеры и подборки фильмов
├── catalog.html        # Каталог: сетка всех фильмов с фильтрацией (сериалы, топы, новинки)
├── movie.html          # Карточка фильма: подробная информация, актеры и похожие фильмы
├── watch.html          # Плеер: страница просмотра (поддерживает VK Video и прямые ссылки)
├── profile.html        # Личный кабинет: история просмотров и данные пользователя
│
├── css/                # Стили (разделены по логике для удобства)
│   ├── variables.css   # Глобальные цвета, шрифты и общие анимации
│   ├── header.css      # Стили шапки, навигации и поисковой строки
│   ├── player.css      # Оформление видеоплеера и кнопок управления
│   ├── components.css  # Общие элементы: карточки, кнопки, бейджи (рейтинг, качество)
│   ├── hero.css        # Стили главного баннера на главной странице
│   └── pages.css       # Layout (разметка) специфичных страниц (каталог, профиль)
│
├── js/                 # Логика (JavaScript)
│   ├── data.js         # База данных: массив объектов со всеми фильмами и их данными
│   ├── render.js       # Движок отрисовки: создает HTML-карточки из данных в data.js
│   ├── player.js       # Управление плеером: инициализация Plyr и работа с VK
│   ├── router.js       # Навигация: обработка URL-параметров (например, ?id=...)
│   ├── app.js          # Главный файл: запускает инициализацию страницы после загрузки
│   └── authgoogle.js   # Авторизация: Firebase Google Auth и управление профилем
│
├── package.json        # Конфигурация проекта и список зависимостей (Firebase)
└── README.md           # Документация проекта с описанием функций
```
## Быстрый старт

1. Распакуй архив
2. Открой `index.html` в браузере — всё работает без сервера
3. Чтобы добавить реальное видео, замени ссылки в `watch.html`:
   ```html
   <source src="https://cdn.example.com/movie.mp4" type="video/mp4" />
   ```
   И в кнопках серверов:
   ```html
   <button class="source-btn" data-src="https://cdn.example.com/movie-1080.mp4">
   ```

## Мини-плеер (нижний)

Плавающий плеер появляется на **всех страницах** автоматически.
- На `watch.html` — связан с основным Plyr-плеером
- На остальных страницах — показывает последний просмотренный фильм из `sessionStorage`
- Кнопка ✕ скрывает плеер

## Настройки плеера на watch.html

В правой колонке:
- Автоматическое качество
- Выбор субтитров
- Выбор дорожки озвучки
- Автопродолжение

## Добавление своих фильмов

Редактируй `js/data.js` — добавляй объекты в массивы `movies` или `series`:

```js
{
  id: 'my-movie',
  title: 'Мой фильм',
  titleOrig: 'My Movie',
  year: 2024,
  duration: '120 мин.',
  rating: 8.2,
  quality: '4K UHD',
  age: '16+',
  genres: ['Драма', 'Триллер'],
  director: 'Режиссёр',
  cast: ['Актёр 1', 'Актёр 2'],
  country: 'Россия',
  lang: 'Русский',
  desc: 'Описание фильма...',
  poster: 'https://ссылка-на-постер.jpg',
  bg:     'https://ссылка-на-фон.jpg',
  video:  'https://cdn.example.com/my-movie.mp4',
  badge: 'new', // null | 'new' | 'hot' | 'top' | 'hd'
}
```
